# Model 模块概述

`model` 模块定义了 `sealdice-core` 应用中使用的所有核心数据结构。这些结构体用于在应用的不同部分之间传递数据，以及将数据存储到数据库中。

## 主要数据模型

- **`PlayerInfo`**: 玩家信息。
- **`GroupInfo`**: 群组信息。
- **`BanListInfoItem`**: 封禁列表条目。
- **`LogOneItem`**: 单条日志条目。
- **`EndPointInfo`**: 连接端点信息。
- **`CensorLogItem`**: 内容审查日志条目。
- **`AttributesItem`**: 属性条目。
